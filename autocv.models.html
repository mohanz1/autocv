ShapeList¶

class autocv.models.shape_list.ShapeList(shape_cls: type[T], data: ndarray[Any, dtype[uint64]] | Sequence[ndarray[Any, dtype[uint64]]] | Sequence[Sequence[int]], center_of_bitmap: Point)[source]¶

Represents a list of shapes including points, rectangles, and contours.

This class provides functionality to manage a collection of shape objects. It allows for accessing individual shapes, retrieving the length of the list, and ordering the shapes based on various criteria defined in the OrderBy enum.

shape_cls¶

The class type of the shapes in the list, such as Point, Rectangle, or Contour.

Type:

type[T]

data¶

The shape data as an array-like structure. Can be an ndarray or a sequence of ndarrays.

Type:

npt.NDArray[np.uintp] | Sequence[npt.NDArray[np.uintp]] | Sequence[Sequence[int]]

center_of_bitmap¶

The center point of the bitmap, used for calculating distances in certain orderings.

Type:

Point

The first shape in the list.

T
